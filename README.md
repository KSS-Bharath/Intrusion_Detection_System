# Intrusion_Detection_System

The project focuses on developing a machine learning-based intrusion detection system to identify malicious network traffic. It begins with collecting and loading the NSL-KDD dataset, followed by data preprocessing where categorical features are encoded and numerical features are scaled. The processed data is then used to train machine learning models for both binary and multiclass classification. The system is evaluated using performance metrics such as accuracy, precision, recall, and confusion matrix. To demonstrate practical applicability, a real-time simulation module is implemented that processes data sequentially and generates alerts when an attack is detected. Additionally, a graphical user interface is developed to allow user interaction and visualization of predictions, making the system user-friendly and suitable for real-world scenarios.

Models Used in the Project:
The project uses two ensemble machine learning models: Random Forest and XGBoost. Random Forest is a bagging-based algorithm that builds multiple decision trees and combines their outputs to improve stability and reduce overfitting. It is used for both multiclass and binary classification in the project. XGBoost is a boosting algorithm that improves prediction performance by sequentially correcting the errors of previous models. It is specifically used for binary classification to achieve better detection performance. Both models are well-suited for handling high-dimensional network data and provide reliable intrusion detection results.
